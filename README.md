# Инструкция по запуску автоматизированных тестов
## 1. Требования
- Python 3.8 или выше
- Установленный `pip`
 
## 2. Установка зависимостей
Откройте терминал в папке с проектом и выполните:
pip install pytest requests
 
## 3. Запуск тестов
pytest test_api.py -v

## 4. Ожидаемый результат
Все тесты должны завершиться успешно
